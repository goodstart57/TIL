# String in PHP

## Length of String

```php
<?php
echo strlen("Hello world!");
?>
```

```
12
```



## The Number of Words

```php
<?php
echo str_word_count("Hello world!");
?>
```

```
2
```



## Reverse a String

```php
<?php
echo strrev("Hello world!");
?>
```

```
!dlrow olleH
```



## Search position of specific word

```php
<?php
echo strpos("Hello world!", "world");
?>
```

```
6
```



## Replace String

```php
<?php
echo str_replace("world", "JaeSeo", "Hello world!");
?>
```

```
Hello JaeSeo
```
